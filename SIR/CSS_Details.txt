CSS - Cascading Style Sheet

Can be implement in 3 ways.
1. Inline
	
	<table style="width:500px;">

	</table>
	
2. Internal
	Same HTML document head portion <style></style>

3. External
	additional file called style.css
	<link  href="css link"/>


CSS Rules
-------------

selector{
	property:value; // css rule
}

: = colon
; = semi-colon

3 types of main selector
------------------------

1. html tag selector	h1,p, 
2. class selector	.classname - class can be added multiple
3. id selector		#idname
